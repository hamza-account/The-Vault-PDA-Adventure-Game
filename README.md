# The-Vault-PDA-Adventure-Game
The Vault is an interactive web-based adventure game inspired by Pushdown Automata (PDA) concepts from the Theory of Computation. Built using HTML, CSS, and JavaScript, the game combines finite-state transitions, stack-based mechanics, and an engaging storyline where players infiltrate a high-security vault by making strategic decisions.

✨ Features
Interactive story-driven gameplay
Pushdown Automata (PDA) simulation
Stack-based inventory system
Multiple game states and transitions
Dynamic room navigation
Alarm and security level system
Real-time PDA state tracking
Transition trace log
Accept and Reject game states
Responsive cyberpunk-themed interface
Restart and replay functionality
🎮 Gameplay

Players take on the role of a skilled infiltrator attempting to break into a highly secured vault. Every action updates the PDA state and stack, requiring players to collect the correct items, unlock restricted areas, and make the right decisions to successfully escape without being caught.

🎯 Project Objective

The objective of this project is to provide an engaging visualization of Pushdown Automata by transforming theoretical concepts into an interactive game. It helps students understand PDA state transitions, stack behavior, and automata principles through real-time gameplay.

📂 How to Run
Clone this repository.
Open the project folder.
Launch index.html in any modern web browser.
Follow the on-screen actions to navigate through the vault.
Reach the Accept State by making the correct decisions, or face the Reject State if security catches you.
